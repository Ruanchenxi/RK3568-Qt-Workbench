##### **2.2.1 普通票的票序列格式**

|名称|字节数|备注|
| - | - | - |
|设备内码|2|也即统称的锁码。|
|动作类型|1|拉开/合上， 二次项、危险点等|
|操作状态|1|0-未操作，1-已操作。|
|显示项地址|4||
注:1.设备内码(锁码)范围为0x0000~0xFFFFF，其中0XFF00-0xFFFF有特殊意义，其中0x0FFE用作跳步码，其他表示意义待定。

2. 操作票序列的步奏数在操作票信息中设定。
2. 当动作类型定义为二次项、危险点或语音提示这三种类型时，显示项地址才有意义，此时电脑钥匙根据显示项地址指定的位置提起显示项显示二次项或危险点的提示内容，另外，当动作类型定义为这三种类型时，设备内码无意义，在做数据时建议设置为0xFFFF。
2. <a name="_hlk380658488"></a>动作类型对应：

   |动态描述|当前状态|目标状态|动作类型值|图形操作后状态|备注|
   | - | - | - | - | - | - |
   |工作转试验|工作（1）|试验（0）|0|0|合到分|
   |试验转工作|试验（0）|工作（1）|1|1|分到合|
   |试验转检修|试验（0）|检修（2）|2|2|试验转检修(不开锁)|
   |检修转试验|检修（2）|试验（0）|3|0|检修转试验(不开锁)|
   |工作转检修|工作（1）|检修（2）|4|2|工作转检修|

   小车动作类型值特殊说明：**动作类型值2，3已用于三态小车不开锁操作步骤(见下面示例：7,2,将\*由试验位置拉出到检修位置   //不开锁  7,3,将\*由检修位置推入到试验位置   //不开锁)，其它类型的设备不能再使用2，3动作类型值。同时，小车没有工作到检修的操作。**

   其它动作类型值：

   备用：5

   备用：6

   验电：验分（也称验无电）0，验合（也称验有电）1

   检查项(二次项（也称纯术语项），不用开锁，直接确认操作)：7

   危险点项：8

   语音提示项（即纯术语项且操作到当前步骤默认有语音播报提示）：9

   **注意1：如果设备安装的是验电锁和位置检测锁，则动作类型值必须为0或1（验分或检分动作类型值为0，验合或检合动作类型值为1）。设备安装其它类型的锁，则动作类型不作限制。**

   **注意2：动作类型值检查项7、危险点项8、语音提示项9,三个值不能用做其它设备类型的动作类型值，已被占用。可以使用0A~FF等动作类型值。**

   **动作类型值取值范围为：0x00~0xFF。**

0. ### **自维护开锁票**
   日常站巡检维护时使用的票，操作票特征如下：

   1）步骤操作无序；

   2）步骤操作不限次数；

   3）钥匙只能操作机械锁设备（其他锁类型不允许操作）；

   4）上位机软件要保证钥匙有且只有一张开锁票；

   5）操作回传后图形不变位。

   **注：开锁票操作发送给钥匙的动作类型值必须是“从合到分”的动作类型值**，否则会出现钥匙显示的术语不对的情况。

0. ### **检修开锁票**
   检修开锁票用于检修隔离任务的**检修操作权**的开锁操作。

   操作票特征如下：

1. 只能使用检修钥匙操作；
1. 无序操作；
1. 不限制设备的操作次数；
1. 钥匙只能操作机械锁；
1. 钥匙操作回传后，图形不变位；
1. 该票类型限制为只能传送到本检修任务绑定的钥匙中（检修任务在传布防票或确认不传布防票时会绑定钥匙A，在传本检修任务的开锁票时，也只能传票到钥匙A）；
1. 钥匙上术语显示为“操作XXX”；
1. 支持设置任务操作期限。
